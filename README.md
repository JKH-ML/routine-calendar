# Routine Calendar

Google Calendar 일정 위에 “루틴” 경험을 얹은 캘린더입니다. 세 가지 루틴 이모지로 하루를 체크하고, 모두 달성하면 반짝이는 테두리로 스스로에게 작은 보상을 줄 수 있습니다.

## 루틴 경험
- **루틴 태깅**: 일정 제목 앞에 `#✏️`, `#🔥`, `#🚀`을 붙여 각 루틴을 표시합니다.
- **완주 보상**: 한 날짜에 세 가지 루틴이 모두 있으면 해당 날짜 셀에 iridescent(무지개빛) 테두리가 나타나며 “완주했다”는 기분 좋은 피드백을 줍니다.
- **캘린더 흐름**: 월 이동, 오늘로 점프, 동기화, 일정 추가/수정/삭제가 자연스럽게 이어지도록 WebView2 UI와 WPF 호스트가 함께 동작합니다.

## 간단 예시
- `#✏️ Morning writing`
- `#🔥 Workout`
- `#🚀 Side project`

세 가지를 같은 날짜에 갖추면 달력이 반짝이며 “오늘의 루틴 완료”를 알려줍니다.
